Como configurar
  1. Acesse console.firebase.google.com
  2. Crie um projeto → adicione um app Web
  3. Ative o Realtime Database
  4. Cole as credenciais acima e salve
  5. Aplique as regras no painel do Database
  6. Inicie o rastreamento no Dashboard
